# wiki-ingest

raw 소스 1개(또는 폴더)를 읽어 wiki에 누적 통합한다.

## 시작 전
1. **`WIKI_SCHEMA.md`를 먼저 읽고 그 규칙을 그대로 따른다.** (구조·언어·frontmatter·설계 결정의 단일 출처)

## 절차
1. 대상 `raw/` 파일(폴더)을 읽는다. 어느 소스인지 불명확하면 사용자에게 확인.
2. 등장하는 **개념·인물·도구**를 추출한다.
3. 각 항목마다:
   - **병합 우선** — `wiki/concepts·people·tools/`에 해당 페이지가 이미 있으면 새로 만들지 말고 **갱신**, 없으면 생성.
   - **모순 보존** — 기존 내용과 출처가 충돌하면 덮어쓰지 말고 페이지 안 `## 모순/주의` 섹션에 양쪽을 출처와 함께 남긴다.
   - frontmatter(`type`/`created`/`updated`/`sources`) 채우고, 본문에서 원본을 `[[ep0X-...]]`로 인용.
4. 관련 페이지끼리 `[[상호링크]]` 연결, `wiki/index.md` 해당 카테고리에 한줄 요약으로 등록.
5. `wiki/log.md`에 `## [YYYY-MM-DD] ingest | <제목>` + 생성/갱신한 페이지 목록 기록.

## 완료 보고
- 만든 페이지 / 갱신한 페이지 / 발견한 모순을 사용자에게 요약.
